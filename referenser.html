Webbproduktion

Nedan presenteras ett urval av de webbplatser jag designat och kodat, utan någon speciell inbördes ordning.

 

Roosweb.se, ett hobbyprojekt jag i många år arbetade med och drev, nu lagt på is. Ett nöjescommunity där spel, bilder, filmklipp med mera delades och publicerades av sidans medlemmar. Sidan innehöll även ett forum, profiler, gallerier, klotterplank med mera.
Tekniker: XHTML 1.0 Transitional, CSS 2.1, ASP 3.0 och en MySQL-databas. Ett fiktivt projekt jag arbetade med för att utöka mina kunskaper i HTML5 samt CSS3.
Tekniker: HTML5, CSS3, ASP.NET 4.0 C#. Ett projekt där jag arbetade med Google Maps API:et för att sammanställa kartor som uppvisade olika handelsplatser inom Borlänge centrum. Ett projekt för att sätta mig in i och arbeta med Google Maps API:et.
Tekniker: HTML5, CSS3, ASP.NET 4.0 C#, JavaScript. Satte upp en webbplats för organisationen Mwambike Foundation som arbetar med föräldralösa barn i Uganda. Sidan drivs med Wordpress på ett av mig modifierat tema. Humornästet var en site jag drev mellan 2009-2012 vars huvudfokus var en yngre målgrupp mellan 12 - 25 år. Webbplatsen var en nöjessite med fokus på roliga filmklipp, bilder och spel. Arbetade mycket med att integrera webbplatsen mot Facebook, Twitter och andra sociala medier.
Tekniker: XHTML 1.0 Strict, CSS 2.1, JavaScript, jQuery, AJAX, ASP 3.0 och en MySQL-databas. Qnote var ett projekt för eget bruk där jag slipade på jQuerykunskaperna tillsammans med HTML5, CSS3, C# och responsiv design. Det var en anteckningsapplikation för dator och mobil där alla anrop och funktioner styrdes med jQuery/AJAX. Tanken var minimalism, alltså så snabbladdad och lightweight som bara möjligt.
Tekniker: HTML5, CSS3, JavaScript, jQuery, AJAX, ASP.NET C#. En guide- och nyhetssite inriktad mot surfplattor, med fokus på kinaplattorna. Drevs med Wordpress och PHP-BB, större skara besökare och mycket uppskattad fram till -12 då jag av tidsbrist valde att lägga ned den. Projektarbete från gymnasietiden där jag och 3 till arbetade med en site och webbtjänst där man kunde sälja sin egenskapade mjukvara. Som ett Blocket eller Tradera för mjukvaruutvecklare. Lades ned -10 i samband med att vårt Ung Företagsamhetsprojekt nått sitt slut.
Tekniker: XHTML 1.0 Strict, CSS 2.1, ASP 3.0, JavaScript och kördes mot en MySQL-databas. Blev kontaktad och ombedd att designa och bygga en webbplats åt Fjälläventyret i Idre.
Tekniker: XHTML 1.0 Strict, CSS 2.1, ASP 3.0 och kördes mot en MySQL-databas. Fiktivt projekt för en gymnasiekurs i webbdesign.
